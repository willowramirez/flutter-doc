# 本地部署

## 打包

```bash
flutter build apk
```

## 上传安卓包到设备

```bash
scp 'build/app/outputs/flutter-apk/app-release.apk' smyze@192.168.42.1:/home/smyze/apk
```

## 连接主机

```bash
ssh [ip]
```

## 连接设备上的安卓设备

```bash
adb connect android:5566
```

## 安装apk到安卓设备

```bash
adb -s android:5566 install -g ~/apk/app-release.apk
```
